# 技术与应用调研
## 主题
- 区块链与人工智能的结合
- 可选方向：
  - 区块链 + Agent
  - 区块链 + LLM
  - 区块链 + RL
  - AI 算力与共识机制

## 调研要求：
- 每组完成 1 个调研，对象需是以下之一：
  - 区块链项目 / 实际应用系统
  - 中文核心刊物论文
  - 国际刊物论文
  - 学术会议级别及以上论文；

- 需撰写调研笔记，最终报告篇幅不少于 20 页。

## 评分规则：
- 基础要求：调研对象数量不符合要求则不得分；
- 评分维度（百分制）：
  - 选题的先进性 / 实用性；
  - 对调研内容的理解把握程度；
  - 调研报告的撰写质量；
  - 最终得分依据调研笔记的评价细则得出。

# 项目评分
- 评分项有 5 项：
  1. 选题的意义与价值；
  2. 研究方法的科学性、合理性；
  3. 实验设计或原型系统设计、实施与结果分析；
  4. 成果的先进性；
  5. 成果的应用价值。
