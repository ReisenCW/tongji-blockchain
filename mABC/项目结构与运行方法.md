## 运行：
```
cd mABC
python main/main.py
```
结果输出在answer.json, 日志输出在output.log

## 数据集获取位置
https://github.com/FudanSELab/train-ticket/
获取数据后需要用handle/metric_generate.py和trace_generate.py生成文件放到data/metric和data/topology下，目前用的是simple_sample文件夹复制过去的简单示例数据。
